## Codebase Patterns
- Tests use `unittest.mock.patch("socket.getaddrinfo")` to simulate DNS resolution without network calls
- Ruff S104 ("Possible binding to all interfaces") fires on "0.0.0.0" string literals in tests ŌĆö suppress with `# noqa: S104`
- No mypy in the venv; use `uv run ruff check` for static analysis. Full type pass = ruff passes.
- `uv run pytest` is the test runner (pyproject.toml configures testpaths: tests)
- To patch a module-level attribute (e.g. `httpx`) in tests, it must be imported at module level in the source file, not inside a function. Lazy imports inside functions cannot be patched as `module.attr`.
